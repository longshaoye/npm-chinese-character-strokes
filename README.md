两万汉字笔顺：[NPM 链接](https://www.npmjs.com/package/chinese-character-strokes)

## 功能

数据来源：https://github.com/CNMan/UnicodeCJK-WuBi 中的 CJK 部分。末尾无笔顺数据的字已删去。

下面是数字 1-5 对应的笔画：
```
U+4E00,一,1

U+4E28,丨,2
U+4E85,亅,2

U+4E3F,丿,3

U+4E36,丶,4
U+4E40,乀,4
U+4E41,乁,4

U+4E59,乙,5
U+4E5A,乚,5
U+4E5B,乛,5
```

API 请参考单元测试
